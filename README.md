# Development

Download this repo, install ruby (2.6.5 recommended) via rbenv, 
`bundle --binstubs`
`bin/lita`

Then send a command like below:
 
# Usage

bot: hh
bot: happyhour
bot: happyhr
!hh

And it will return happy hour ideas.
